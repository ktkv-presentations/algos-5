Определение и вызов подпрограмм

Подпрограмма

независимые блоки кода, которые выполняют определённые задачи

Функция

независимый блок кода, предназначенный для выполнения определённой задачи

Метод

функция, которая "принадлежит" объекту или классу

Функции и методы позволяют

Упрощать разработку

вместо того чтобы многократно писать один и тот же код, можно использовать функцию

Повышать читаемость кода

можно выделить ключевые моменты в программе и улучшить понимание её структуры

Повторное использование кода

можно использовать в других частях программы или даже в других проектах

Модульность

позволяют разделять код на независимые компоненты, которые можно разрабатывать, тестировать и модифицировать отдельно

Преимущества использования подпрограмм

Снижение дублирования кода

один и тот же код не нужно писать несколько раз, он будет храниться в функции

Улучшение отладки и тестирования

отдельные функции легче тестировать и отлаживать, чем большие блоки кода

Гибкость и расширяемость

используя параметры и аргументы, можно адаптировать функцию для решения различных задач, не изменяя её внутренний код

Читаемость и поддерживаемость

функция с понятным именем и задачей делает код более понятным для других разработчиков

Без использования функций


print("Привет, Алексей!")
print("Привет, Анна!")
print("Привет, Дмитрий!")

С использованием функций


def greet(name):
    print(f"Привет, {name}!")

greet("Алексей")
greet("Анна")
greet("Дмитрий")

Функции

определяется с использованием ключевого слова def


def имя_функции(параметры):
		# тело функции
		return результат

Параметры функций

Параметры функций задаются при её определении. Аргументы, передаваемые функции при вызове, соответствуют этим параметрам


def имя_функции(параметры):
		# тело функции
		return результат
где name и message — параметры, а "Алексей" и "Добро пожаловать" — аргументы.

Обязательные и необязательные параметры

Обязательные параметры

параметры, которые должны быть переданы при вызове функции. Если они не передаются, возникнет ошибка

Необязательные параметры

параметры с заданными значениями по умолчанию. Если аргумент не передан, используется значение по умолчанию

Пример с параметрами по умолчанию


def greet(name, message="Привет"):
    print(f"{message}, {name}!")

greet("Алексей")  # Вывод: Привет, Алексей!
greet("Анна", "Добрый день")  # Вывод: Добрый день, Анна!
message имеет значение по умолчанию "Привет", которое используется, если этот аргумент не указан при вызове функции

Пример создания и вызова функции

Определим функцию, которая вычисляет площадь прямоугольника, и продемонстрируем её вызов:

Пример с параметрами по умолчанию


def calculate_area(length, width):
    area = length * width
    return area

# Вызов функции с аргументами 5 и 3
result = calculate_area(5, 3)
print(result)  # Вывод: 15

Отличие между функцией и методом

Функции

могут быть определены и вызваны независимо, они не привязаны к какому-либо объекту

Методы

функции, которые принадлежат объектам или классам. Они вызываются через экземпляр объекта, и часто могут изменять его состояние

Пример методы в классе


class Calculator:
    def add(self, a, b):
        return a + b

calc = Calculator()
print(calc.add(2, 3))  # Вывод: 5

Область видимости переменной

Область видимости

часть программы, в которой переменная доступна для использования

Локальная область видимости

переменные внутри функции

Глобальная область видимости

Глобальная область видимости

Локальные переменные

Локальные переменные определяются внутри функций и доступны только внутри этих функций


def my_function():
    x = 10  # Локальная переменная
    print(x)

my_function()  # Вывод: 10
print(x)  # Ошибка: переменная x не существует в глобальной области

Глобальные переменные

Глобальные переменные объявляются вне функций и доступны во всей программе. Они могут использоваться как внутри функций, так и за их пределами


x = 5  # Глобальная переменная

def my_function():
    print(x)  # Доступ к глобальной переменной внутри функции

my_function()  # Вывод: 5
print(x)  # Вывод: 5

Изменение глобальных переменных внутри функции

Чтобы изменить глобальную переменную внутри функции, нужно использовать ключевое слово global


x = 10

def modify_global():
    global x  # Объявляем, что x — глобальная переменная
    x = 20  # Изменяем глобальную переменную

modify_global()
print(x)  # Вывод: 20

Вложенные функции и замыкания

замыкания

функции, которые "запоминают" переменные, определённые в их внешней функции

def outer_function():
    x = 10
    def inner_function():
        print(x)  # Доступ к переменной внешней функции
    inner_function()

outer_function()  # Вывод: 10
